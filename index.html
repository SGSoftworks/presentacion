HTTP/HTTPS

El idioma de la World Wide Web

¿Qué es?

Son las reglas que usa tu navegador para conectarse a un sitio web y transferir la información (texto, imágenes, videos). HTTPS es la versión segura que encripta todos los datos.

Puertos:

• HTTP: 80 (Para sitios normales)

• HTTPS: 443 (Para sitios seguros)

Beneficios y Desafíos

  • • Pro: Es el estándar universal, fácil de usar.
  • • Pro: HTTPS mantiene tu información segura.
  • • Contra: HTTP es inseguro, se envía en texto plano.
  • • Contra: Las conexiones pueden ser lentas si no están optimizadas.

SSH

Tu llave secreta para la conexión remota

¿Qué es?

Es un protocolo para acceder de forma segura a otros computadores a través de Internet. Permite controlar un servidor a distancia como si estuvieras sentado frente a él.

Puerto:

• 22 (Tu puerta segura para entrar)

Beneficios y Desafíos

  • • Pro: Toda la información está cifrada y protegida.
  • • Pro: Te permite transferir archivos de forma segura.
  • • Contra: Si no usas contraseñas fuertes, puede ser vulnerable.

FTP

La herramienta para mover archivos grandes

¿Qué es?

Es la forma estándar de subir y descargar archivos entre tu computador y un servidor. Es ideal para mover muchos archivos a la vez, como los de un sitio web.

Puertos:

• 21 (Para el control de la conexión)

• 20 (Para la transferencia de datos)

Beneficios y Desafíos

  • • Pro: Es muy eficiente para grandes cantidades de datos.
  • • Pro: Puedes reanudar una descarga si se interrumpe.
  • • Contra: No es seguro por defecto, la información viaja sin protección.

DNS

El director de la agenda telefónica de Internet

¿Qué es?

Traduce los nombres de los sitios web (como `google.com`) a direcciones IP (como `172.217.10.14`). Sin el DNS, tendrías que memorizar números para cada sitio.

Puerto: 53

Es la puerta por donde viajan las búsquedas

Beneficios y Desafíos

  • • Pro: Hace que Internet sea fácil de usar para las personas.
  • • Pro: Es muy rápido gracias a su sistema de caché.
  • • Contra: Puede ser vulnerable a ataques que te desvían a sitios falsos.

SMTP

El cartero que entrega tus emails

¿Qué es?

Es la regla principal para enviar correos electrónicos de un servidor a otro. Funciona como un servicio de mensajería que asegura que el email llegue a su destino.

Puertos:

• 25 (Envío normal)

• 465 (Envío seguro)

• 587 (Envío más moderno)

Beneficios y Desafíos

  • • Pro: Es el estándar para enviar emails, es muy confiable.
  • • Pro: Soporta la autenticación para evitar el abuso.
  • • Contra: Es el protocolo más usado para enviar spam.

Puntos Clave

  • • Los puertos son las "puertas" de un servicio.
  • • Los protocolos son las "reglas" para hablar en la red.
  • • La seguridad es clave para proteger tus datos.
  • • Siempre prefiere las versiones seguras de los protocolos.

Mejores Prácticas

  • • Usa HTTPS siempre que sea posible.
  • • Cierra los puertos que no necesites en tu firewall.
  • • Actualiza tu software regularmente.
  • • Usa contraseñas fuertes y únicas.
